Data Visualization: Programming Assignment 2

Interactive force graph shows American college football games from 2000.

Explanation

The graph depicts college football games played in 2000. Each node represents a university, and the edges indicate games. For this assignment, I wanted to explore how interactions could be used to enhance a graph. I chose the football data set for two reasons. First, the 2017 college football season is just starting, so it was interesting to see how conferences have changed over the years. Second, due to the data set's modest size, I could focus on adding interactions, without worrying too much about latency.

Overall it's difficult to follow edges, so I added filtering by school and conference. These filters allow users to explore sections of the graph making it easier to differentiate connections.

For the most part, the nodes naturally grouped up according to their conference. The number of games a school played outside of their conference varied, but every school played at least one.

Data Source

University of California Network Data Repository. (2017). American College Football [Data File]. From https://networkdata.ics.uci.edu/data.php?id=5

Modifications Made to Original Data File

NetworkX, a Python package, was used to convert the original Graph Modelling Language (GML) into JavaScript Object Notation (JSON). In order to facilitate parsing multigraph 1 was added to the header of the downloaded GML data file.
